# kotlin-lotto

### 그라운드 룰
- 내비게이터와 드라이버는 커밋 단위로 교체한다.
- 배려한다.

### 기능 목록
- 로또 구입 금액을 입력받는다.
- 로또 구입 금액이 유효한지 유효성 검사를 진행한다.
- 로또를 1000원당 1장 구매한다.
- 구입한 로또를 생성한다.
- 구입한 로또를 출력한다.
- 지난 주 당첨 번호를 입력받는다.
- 지난 주 당첨 번호가 유효한지 유효성 검사를 진행한다.
- 보너스 번호를 입력받는다.
- 보너스 번호가 유효한지 유효성 검사를 진행한다.
- 당첨 결과를 통계한다.
- 통계한 내용을 출력한다.
